HuBMAP User Interface v0.5.0

Proof-of-Concept User Interface (UI) for the Common Coordinate Framework (CCF) to be delivered by the MC-IU team in June 2019.
The current user interface supports:
  • Visual browsing of tissue samples and metadata at the whole body, organ, tissue, and cell level.
  • Filtering by metadata (age, gender, TMC, and technology), results are presented at all views.
  • Submit questions and comments on the CCF UI.
  • Semantic search by ontology, results are presented at all levels.
  • Data download at the whole body, organ, tissue, and cell level, i.e., link to https://sampledata.hubmapconsortium.org.
